Project Description:
BloodDonationMVC is a web-based Blood Donation Management System developed as an academic project using MVC architecture. The system helps users to register as blood donors and allows others to search for donors based on blood group and city.
The project demonstrates the use of JSP, Servlets, JDBC, and MySQL for building a simple and efficient backend-driven web application.


Technologies Used:
Frontend:
- HTML
- JSP
- CSS

Backend:
- Java (Servlets, JDBC)

Database:
- MySQL

Server:
- Apache Tomcat

IDE:
- NetBeans


Project Structure:
- src/            : Java source code (Controller, Model, DTO, DB)
- WebContent/     : JSP pages and configuration files
- database/       : MySQL database file (blooddonation.sql)
- lib/            : MySQL Connector JAR
- README.txt      : Project documentation


Main Features:
- Donor Registration
- User Login
- Search Blood Donors by Blood Group and City
- MVC Architecture Implementation
- Database Connectivity using JDBC


How to Run the Project:
1. Install JDK (Version 17 or above)
2. Install Apache Tomcat Server
3. Import the project into NetBeans
4. Import the database using blooddonation.sql
5. Configure MySQL username and password in DBConnector.java
6. Run the project on Tomcat server
7. Open index.html in browser


Database Information:
Database Name: blood_donation
Table Name: donors


Note:
This project is developed purely for academic learning purposes and demonstrates basic CRUD and MVC concepts.
